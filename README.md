# PyMOA
Pythonic wrapper around Massive Online Analysis [MOA](https://github.com/Waikato/moa)

## Supported Algorithms

1. Stream Clustering

	1. Clusterers [DenStream using DBSCAN]
	2. Clustree
	3. Denstream
	4. Dstream
	5. StreamKM

## Supported Metrics

1. F1 Score
2. Purity
